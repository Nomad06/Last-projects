-На вход(через аргумент командной строки) подается имя файла <ClassName>.java с классом, наследующимся от ListResourceBundle и содержащим инициализатор массива
пар строк (каждая пара – массив из двух элементов, т.е. имеет вид {"ключ", "значение"}). На выходе программа должна создать файл <ClassName>.properties, 
читабельный с помощью PropertyResourceBundle, т.е. содержащий строки в формате "ключ=значение". Считается, что каждая пара "ключ-значение" 
расположена в одной строке файла *.java. 
-Поддерживается запись в файл *.properties символов, не относящихся к набору ASCII (например, русских символов), в формате "\uXXXX", 
где XXXX – шестнадцатеричное представление юникод-символа.
-Также копируются комментарии к строкам класса (только те, которые встречаются после строк {"ключ", "значение"}) в файл *.properties 

Вводите полный путь к файлу через командную строку.
