   
   Касательно SimpleXML:
   
   С помощью DOM API в Java-коде создать XML документ вида "<tagName>textNode</tagName>".
   Требования:
 - Результат должен быть корректным (well-formed) XML документом.
 - Никаких переводов строк или других дополнительных символов не должно быть добавлено в textNode.
	
	 С помощью SAX API проверяется, что во входящем потоке содержится корректный (well-formed) XML-документ.
   В качестве результата возвращается имя корневого элемента документа, а в случае ошибки (если документ не well-formed) выбрасывается SAXException.
	

	Касательно XPathCaller:
	
	Работа с XPath выражениями для получения данных из XML документов в папке с проектом
